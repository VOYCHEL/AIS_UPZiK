# АИС «УПЗиК» — Управление производственными заказами и комплектацией

**Назначение системы:**  
Автоматизированная информационная система для автомобильного завода ООО «АвтоПромИндустрия».  
Система предназначена для:
- Планирования производства (формирование сменных заданий, расчёт такта конвейера);
- Учёта комплектующих на складе и резервирования под заказы;
- Отслеживания статуса сборки автомобилей по этапам (сварка, окраска, сборка);
- Регистрации отклонений (простои, брак, нехватка материалов);
- Формирования отчётности для руководства и экономистов.

**Основные модули:**
1. Модуль планирования
2. Модуль складского учёта
3. Диспетчерский модуль
4. Модуль аналитики и отчётности

**Технологический стек:**
- Бэкенд: Python (FastAPI)
- Фронтенд: React
- СУБД: PostgreSQL
- Контейнеризация: Docker
- Система контроля версий: Git
